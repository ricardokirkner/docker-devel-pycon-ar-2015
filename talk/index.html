El problema

Una aplicación web moderna está tiene una arquitectura compleja

  • Web Server (apache, nginx)
  • WSGI Server (gunicorn, uwsgi)
  • Application (django, flask)
  • Database (postgresql, mysql)
  • Cache Server (memcached, redis)
  • Asynchronous Tasks (celery)
  • Email Server (postfix, exim)
  • Error Reporting (sentry)
  • Logging (logstash, elasticsearch, kibana)
  • Metrics (graphite, statsd)
  • Monitoring (nagios, zabbix)
